# 振霊恐室
このゲームはAmptixを使用したホラーゲムです。
ビルドデータを再生するにはAmptixのライセンスが適応されているPCで再生する必要があります。
また、このリポジトリには以下に記載している再配布不可能なアセットを含めていません。

# 開発期間
